# 🛠️ Sistema de Help Desk

Um sistema robusto e responsivo de gerenciamento de chamados (tickets) focado em otimizar o fluxo de atendimento e a gestão de usuários. Desenvolvido para fins de aprendizado e aprimoramento em back-end e banco de dados relacional.

## ✨ Funcionalidades

* **Painel Administrativo:** Gestão completa de usuários (criação, exclusão e edição de permissões) e dashboard com estatísticas, incluindo o tempo médio de resolução dos chamados.
* **Gestão de Tickets:** Modal centralizado em "Ver Detalhes" para visualização rápida, edição de chamados abertos e controle do processo de resolução.
* **Comunicação Direta:** Chat integrado diretamente nos tickets para interação entre a equipe e o cliente.
* **Segurança:** Autenticação de usuários com senhas criptografadas utilizando Bcrypt.
* **Experiência do Usuário (UX/UI):** * Design totalmente responsivo.
  * Menu lateral intuitivo e filtros de busca dinâmicos.
  * **Dark Mode** persistente (salva a preferência do usuário).

## 🚀 Tecnologias Utilizadas

* **Back-end:** Node.js com framework Express.
* **Banco de Dados:** SQLite (banco relacional leve e embutido).
* **Segurança:** Bcrypt (hash de senhas).
* **Front-end:** HTML, CSS e JavaScript (com responsividade e alternância de temas).
